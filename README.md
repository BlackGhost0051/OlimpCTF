# Start project

## Frontend

**Framework:** Angular

To run the frontend:
```bash
  cd Frontend
  ng serve
```

## Backend

**Framework:** Express.js

To run the backend API:
```bash
  cd Backend/api
  npm install
  npm run start
```

## Docker

To run the full stack using Docker:
```bash
  docker-compose up --build
```
